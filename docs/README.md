## 🔧 技术概览

### 核心技术栈
```
前端框架: Streamlit 1.29.0
后端语言: Python 3.13
数据库: SQLite 3
可视化: Plotly + Open3D
数据处理: NumPy + Pandas
```

### 支持的数据格式
```
点云数据: .pcd, .las/.laz, .txt/.xyz
图像数据: .png, .jpg, .jpeg
配置文件: .yaml, .yml
传感器数据: .csv, .json
ROS数据: .bag
```

### 核心功能模块
```
├── 用户认证系统
│   ├── 注册登录
│   ├── 会话管理
│   └── 安全防护
├── 数据管理系统
│   ├── 文件上传
│   ├── 数据组织
│   └── 搜索过滤
└── 可视化系统
    ├── 点云渲染
    ├── 交互分析
    └── 对比展示
```

## 📖 使用指南

### 文档阅读建议

#### 🚀 快速上手 (15分钟)
1. 阅读 [架构设计文档 → 1. 项目概述](./架构设计文档.md#1-项目概述)
2. 浏览 [模块划分方案 → 1. 模块总体划分](./模块划分与实现方案.md#1-模块总体划分)
3. 查看项目目录结构和依赖关系

#### 🔧 开发准备 (30分钟)
1. 详读 [架构设计文档 → 2. 系统架构](./架构设计文档.md#2-系统架构)
2. 学习 [模块划分方案 → 2. 详细模块分析](./模块划分与实现方案.md#2-详细模块分析)
3. 了解开发环境配置和测试策略

#### 📚 深入理解 (60分钟)
1. 完整阅读两个文档
2. 理解技术实现细节和设计决策
3. 掌握扩展开发和维护要点

### 开发流程建议

#### 新功能开发
```
1. 查阅架构设计 → 确定技术方案
2. 参考模块划分 → 确定实现位置
3. 遵循编码规范 → 实现具体功能
4. 编写单元测试 → 保证代码质量
5. 更新文档说明 → 维护文档一致性
```

#### 问题排查
```
1. 查看错误日志 → 定位问题模块
2. 参考模块文档 → 理解预期行为
3. 运行相关测试 → 验证功能状态
4. 查阅安全机制 → 排除安全限制
```
